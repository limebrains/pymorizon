# Running

### Installation
Version from https://pypi.python.org/pypi/pymorizon:
```
pip install pymorizon
```
Latest version from git:
```
pip install git+git://github.com/limebrains/pymorizon.git
```

### Example script
```
python example.py
```

### Travis pipeline
```
tox
```

### Tests
```
py.test tests.py -vv --pdb

```
